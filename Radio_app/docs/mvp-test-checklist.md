# MVP Test Checklist (Android)

## Устройства

- Android 8+
- Android 10
- Android 13+

## Винтажный интерфейс

- Экран выглядит как корпус приёмника (фон, шкала, сетка, лампа, ручка); нет цифрового слайдера, года, текстовой панели станции и кнопок Play/Pause/Stop на экране.
- Ручка настройки: вертикальный жест меняет частоту плавно; после отпускания частота привязывается к ближайшей из конфигурации.
- Стрелка на шкале движется плавно в соответствии с настройкой.
- Лампа в углу мягко пульсирует после прогрева.
- При холодном старте 1–2 с затемнение/прогрев, затем эфир по сохранённой частоте.
- Подсказка «Вращайте ручку настройки» исчезает после первого жеста на ручке.

## Функциональные проверки

- Холодный старт без интернета восстанавливает последнюю **частоту** и запускает станцию или шум.
- Пустая частота воспроизводит `static` в бесконечном цикле.
- Переключение частоты всегда делает короткий шум-переход, затем новую очередь (все треки станции по годам подряд) или шум.
- Для станции очередь зацикливается по окончании.
- Возврат на прошлый недоигранный трек не выполняется после переключения (очередь пересобирается заново).

## Фоновое воспроизведение

- При сворачивании приложения звук продолжает играть.
- В уведомлении есть Play/Pause/Stop.
- Тап по уведомлению возвращает в приложение.

## Сценарии устойчивости

- Входящий звонок/потеря аудиофокуса: нет крэшей, аудио корректно возобновляется вручную.
- Убийство Activity: после повторного входа состояние корректно восстанавливается.
- Несколько быстрых переключений частоты: проигрывается только последняя выбранная частота.
