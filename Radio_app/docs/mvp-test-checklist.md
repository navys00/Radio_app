# MVP Test Checklist (Android)

## Устройства

- Android 8+
- Android 10
- Android 13+

## Функциональные проверки

- Холодный старт без интернета запускает станцию/шум по последним сохраненным `frequency` и `year`.
- Пустая частота воспроизводит `static` в бесконечном цикле.
- Переключение частоты и года всегда делает короткий шум-переход, затем запускает новую очередь.
- Для станции играет очередь треков года и зацикливается по окончании.
- Возврат на прошлый недоигранный трек не выполняется после переключения (очередь пересобирается заново).

## Фоновое воспроизведение

- При сворачивании приложения звук продолжает играть.
- В уведомлении есть Play/Pause/Stop.
- Тап по уведомлению возвращает в приложение.

## Сценарии устойчивости

- Входящий звонок/потеря аудиофокуса: нет крэшей, аудио корректно возобновляется вручную.
- Убийство Activity: после повторного входа состояние корректно восстанавливается.
- Несколько быстрых переключений частоты/года: проигрывается только последняя выбранная комбинация.
