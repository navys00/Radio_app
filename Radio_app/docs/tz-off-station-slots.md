# ТЗ: слоты вне станций (`offStationSlots`)

Конфиг: [stations.json](../src/assets/stations.json), поле `offStationSlots`.

## Правила

- Каждый слот: `id`, `minKhz`, `maxKhz`, `years` (частично по годам), опционально `overridesStationCapture` (по умолчанию `true`).
- При `overridesStationCapture: true` слот **имеет приоритет** над захватом станции в своём диапазоне («поймать чужую волну»).
- При `false` слот срабатывает только если частота **не** попадает в радиус захвата ни одной станции.
- Гистерезис выхода из слота: см. `OFF_STATION_HYSTERESIS_KHZ` и `effectiveOffSlotHysteresisKhz` в [radioData.ts](../src/data/radioData.ts) — вертикаль зоны продления ограничивается ближайшими станциями, чтобы не блокировать уверенный приём.

## Приёмка

- На частоте секретного слота (см. `secret_mw_945`) воспроизводится контент слота, а не ближайшая станция.
- При выходе за пределы слота (и гистерезиса) снова действует обычный захват станции или шум.
